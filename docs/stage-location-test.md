# Test modulu Moja lokalizacja / Moj etap

Po wdrozeniu sprawdz na telefonie:

1. Otworz strone szkolenia.
2. W sekcji `Gdzie mam isc?` wybierz `Etap 1`, `Etap 2`, `Etap 3`, `Etap 4`, `Etap 5`, `Etap 6` albo `Office`.
3. Sprawdz, czy pokazuje sie karta z opisem i przyciskiem `Otworz mape`.
4. Zmien jezyk na UA / RU / EN / AZ i sprawdz, czy nazwy i opis zmieniaja sie poprawnie.
5. Przejdz szkolenie i test.
6. Na ekranie ukonczenia sprawdz, czy widac wybrany etap.
7. Kliknij `Kopiuj potwierdzenie` i sprawdz, czy w skopiowanym tekscie jest linia z etapem.

Oczekiwany efekt:

```text
SZKOLENIE UKONCZONE
Imie i nazwisko: ...
Data: ...
Jezyk: ...
Lokalizacja: Siechnice
Etap: Etap 1
```
