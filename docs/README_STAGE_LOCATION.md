# Modul: Moja lokalizacja / Moj etap

Ten modul pomaga pracownikowi szybko sprawdzic, gdzie ma isc pierwszego dnia.

## Jak dziala

1. Pracownik wybiera: Office albo Etap 1-6.
2. System pokazuje opis wybranego miejsca.
3. Pracownik klika `Otworz mape`.
4. Wybor zapisuje sie lokalnie w telefonie.
5. Po ukonczeniu szkolenia etap pojawia sie na ekranie potwierdzenia.
6. Po kliknieciu `Kopiuj potwierdzenie` etap jest dodany do skopiowanego tekstu.

## Pliki

- `stage-location.js` - logika wyboru etapu i kopiowania potwierdzenia z etapem.
- `stage-location.css` - wyglad modulu.
- `data/locations.js` - linki do map biura i wejsc E1-E6.
- `docs/stage-location-test.md` - lista testowa.
