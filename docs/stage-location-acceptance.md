# Kryteria akceptacji

Modul jest gotowy, jezeli:

- widac sekcje `Gdzie mam isc?`,
- mozna wybrac Office albo Etap 1-6,
- po wyborze pokazuje sie karta z opisem,
- przycisk `Otworz mape` prowadzi do poprawnego linku,
- wybor etapu zostaje po odswiezeniu strony,
- etap pokazuje sie na ekranie ukonczenia szkolenia,
- skopiowane potwierdzenie zawiera etap,
- zmiana jezyka aktualizuje teksty modulu.
